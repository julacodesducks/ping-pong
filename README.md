# 🏓 Ping Pong Game

## 📖 About The Project
A classic, retro-style Ping Pong arcade game built with Python. 

The main goal of this project is to practice logic building, handling user inputs in a dynamic environment. The game is currently in a fully playable basic state and is under active development.

## 🛠️ Built With
* Python 3
* Pygame

## ✨ Current Features
* Two-player local multiplayer mode
* Real-time score tracking
* Basic ball collision detection with paddles and walls
* Sound effects for ball bounces and scoring
* Basic main menu
